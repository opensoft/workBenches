# Skill: Internal Communications

## Triggers
- Writing status reports
- Leadership updates
- 3P (Third Party) updates
- Company newsletters
- FAQs
- Incident reports
- Project updates

## Communication Types

### Status Reports
**Structure:**
1. Executive Summary (2-3 sentences)
2. Progress This Period
3. Blockers/Risks
4. Next Steps
5. Metrics (if applicable)

### Leadership Updates
**Structure:**
1. TL;DR (one paragraph)
2. Key Wins
3. Challenges & Mitigations
4. Upcoming Milestones
5. Ask/Support Needed

### Incident Reports
**Structure:**
1. Incident Summary
2. Timeline of Events
3. Impact Assessment
4. Root Cause
5. Remediation Steps
6. Preventive Measures

### Project Updates
**Structure:**
1. Project Status (Green/Yellow/Red)
2. Completed Milestones
3. Current Focus
4. Risks & Mitigations
5. Resource Needs
6. Timeline Update

## Tone Guidelines
- Clear and concise
- Lead with the most important information
- Use bullet points for scanability
- Quantify where possible
- Be honest about challenges
- Always include next steps or asks
